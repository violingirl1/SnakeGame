# 🐍 Java Snake Game

A classic Snake game built using Java Swing.

## ✨ Features

- 🎮 Smooth snake movement
- ⌨️ Keyboard controls
- 🍎 Random food spawning
- 📈 Score tracking
- 🐍 Snake growth
- 💥 Wall collision
- 🔄 Restart functionality
- 🔊 Sound effects

## 🛠️ Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)

## 🚀 How to Run

1. Clone this repository.
2. Compile the Java files.
3. Run `Main.java`.
